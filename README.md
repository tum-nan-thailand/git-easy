# คู่มือเรียนรู้ Git สำหรับการทำงานในทีม

คู่มือนี้จะช่วยให้คุณเข้าใจการใช้งาน Git ในสถานการณ์ต่างๆ ที่มักพบในการทำงานเป็นทีม โดยเฉพาะการจัดการกับ Git Conflict และการใช้คำสั่งขั้นสูงเช่น rebase, reset, stash

## เนื้อหาในคู่มือ

1. [สถานการณ์จำลอง: Git Conflict](scenarios/conflict-scenario.md)
2. [การใช้งาน Git Rebase](guides/git-rebase-guide.md)
3. [การใช้งาน Git Reset](guides/git-reset-guide.md)
4. [การใช้งาน Git Stash](guides/git-stash-guide.md)
5. [แบบฝึกหัดและเฉลย](exercises/README.md)

## วิธีการใช้คู่มือ

1. ศึกษาเนื้อหาในแต่ละหัวข้อตามลำดับ
2. ทดลองทำตามสถานการณ์จำลองในเครื่องของคุณ
3. ทำแบบฝึกหัดเพื่อทดสอบความเข้าใจ
4. ตรวจสอบเฉลยเมื่อทำแบบฝึกหัดเสร็จ

## เริ่มต้นใช้งาน

เริ่มต้นจากการศึกษา [สถานการณ์จำลอง: Git Conflict](scenarios/conflict-scenario.md) เพื่อเข้าใจปัญหาที่มักพบในการทำงานร่วมกัน
